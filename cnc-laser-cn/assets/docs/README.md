## Quick Reference

### CO2 vs Fiber Laser Selection Guide

**Choose CO2 Laser When:**
- Working with non-metals (wood, acrylic, leather, fabric)
- Need high precision engraving
- Working with thinner materials
- Budget is primary concern

**Choose Fiber Laser When:**
- Working with metals (steel, stainless, aluminum)
- Need high cutting speed for thick materials
- Planning industrial production
- Long-term cost efficiency is priority

### Power Selection Guide

**Low Power (20W-60W):**
- Thin materials (<5mm)
- High detail engraving
- Hobby/small business use

**Medium Power (80W-150W CO2 / 1kW-3kW Fiber):**
- Medium thickness materials (5-15mm)
- Production work
- Most common applications

**High Power (150W+ CO2 / 3kW+ Fiber):**
- Thick materials (>15mm)
- High-speed production
- Industrial applications
