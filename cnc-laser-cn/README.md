# CNC激光切割技术有限公司 - 营销网站

## 项目概述

这是CNC激光切割技术有限公司的官方营销网站，展示公司的激光切割服务、企业文化和联系方式。网站采用现代化的响应式设计，专为制造业客户打造。

## 页面结构

### 主要页面
- **首页 (index.html)** - 公司介绍、核心服务、客户案例
- **关于我们 (about.html)** - 企业历史、发展历程、团队介绍、资质认证
- **联系我们 (contact.html)** - 联系方式、在线咨询、工厂位置、常见问题

### 技术特性
- 响应式设计，支持桌面端和移动端
- 现代化CSS3样式，渐变背景和动画效果
- 符合SEO优化的meta标签和结构化数据
- 面包屑导航和页面间链接
- 中文界面，适合中国制造业市场
