## Deployment Instructions

1. **Directory Setup:**
   - Create directory: `rummy-app-en-in/`
   - Place all HTML files in this directory
   - No subdirectories needed

2. **Server Configuration:**
   - Set directory as web root for domain
   - Configure `index.html` as default document
   - Enable gzip compression
   - Set proper MIME types

3. **DNS Configuration:**
   - Point domain `zhyw-china.com` to server
   - Ensure SSL/TLS certificate is installed
   - Configure canonical URLs properly

4. **Search Engine Submission:**
   - Submit sitemap.xml to Google Search Console
   - Submit to Bing Webmaster Tools
   - Configure in Yandex if targeting CIS markets
   - Monitor indexing status
