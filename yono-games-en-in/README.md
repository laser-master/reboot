# Yono Games - English Promotional Website

## Project Information
- **Domain**: https://zeweikeji.com/
- **Language**: English
- **Target Market**: India
- **Year**: 2026
- **Purpose**: SEO-optimized promotional website for Yono Games

## Project Structure

This is a standalone static HTML website for Yono Games, completely isolated in the `yono-games-en-in` directory.

### Pages Created (9 pages + sitemap + robots.txt)

1. **index.html** - Homepage introducing Yono Games platform
2. **yono-app-download.html** - Complete download guide for Android, iOS & PC
3. **yono-games-apk.html** - Android APK download specific page
4. **yono-games-rummy.html** - Yono Rummy features and guide
5. **yono-games-777.html** - Yono Games 777 premium gaming
6. **yono-games-slots.html** - Slots Winner 789 and slot games
7. **yono-games-spin.html** - Yono Games Spin features
8. **yono-app-login.html** - Login guide and account features
9. **yono-app-features.html** - Complete features overview
10. **sitemap.xml** - XML sitemap for SEO
11. **robots.txt** - Search engine crawler configuration
